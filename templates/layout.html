About

A mobile friendly tool to lookup and troubleshoot DNS problems. I developed this tool in May of 2020 in the middle of the Corona virus crisis. Feel free to use and comment on any bug or feature. The tool is using the DNS python library

Dig shotgun

  • Minimal input to find the information needed
  • Parallel execution of lookups for fast results
  • Native Python code
  • Color coding to see differences in responses from different servers
  • Icons to highlight server ownership for top DNS provider
